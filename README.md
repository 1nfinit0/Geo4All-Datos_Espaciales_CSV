# Semana 5: Integración de datos Espaciales y no Espaciales

Esta sesión se centrará en la integración de datos espaciales y no espaciales utilizando herramientas de análisis de datos dentro del entorno de QGIS.

Los recursos utilizados en esta sesión incluyen:

[Recursos espaciales: Cartofy](https://1nfinit0.github.io/cartofy/)

[Recursos no espaciales: Plataforma de Datos abiertos del Perú](https://www.datosabiertos.gob.pe/)

[Fallecidos por COVID-19 en Perú](https://www.datosabiertos.gob.pe/dataset/fallecidos-por-covid-19-ministerio-de-salud-minsa)

[Luminarias Perú](https://www.datosabiertos.gob.pe/dataset/equipo-de-alumbrado-p%C3%BAblico-organismo-supervisor-de-la-inversi%C3%B3n-en-energ%C3%ADa-y-miner%C3%ADa)
